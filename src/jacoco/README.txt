Jars in this folder are used to provide JaCoCo code coverage support
for Java.  The following open-source jars should be placed in this directory:

jacocoagent.jar
jacocoant.jar

Use the latest version from:

http://www.eclemma.org/jacoco/index.html

The current official build of this plug-in uses:

http://search.maven.org/remotecontent?filepath=org/jacoco/jacoco/0.7.7.201606060606/jacoco-0.7.7.201606060606.zip
